트랙 주행, 원하는 차량으로 직접 체험하세요!

Test Drive 프로그램은 BMW, MINI 전 모델 중 고객님께서 원하시는 모델을 직접 선택하여 트랙을 주행해 볼 수 있는 체험 프로그램입니다. 관심있던 모델부터 평소 경험하기 어려웠던 모델까지, 합리적인 가격으로 BMW와 MINI를 직접 경험하세요.
